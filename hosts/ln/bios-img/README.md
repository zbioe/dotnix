# Change bios boot image in Lenovo Thinkpad
Steps to change logo of bios boot image

## Get your serial number

``` bash
sudo dmidecode  -s 'system-serial-number'
```

## Enter in pcsupport.lenovo.com and put your serialnumber to find the right iso
Enter your serial number in body search of
https://pcsupport.lenovo.com/
The page redirects you to the support page of your product

## Download the iso
In the support page go to:
   - Drive & Software
